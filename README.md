📌 Sobre o projeto

Este é um projeto simples de front-end desenvolvido como parte dos estudos iniciais em desenvolvimento web. A proposta foi criar uma landing page temática inspirada no filme Pânico 7, com foco na prática de conceitos fundamentais de HTML e CSS.

A aplicação apresenta um layout organizado contendo pôster, título, informações do filme, sinopse e um botão direcionando para o trailer. O projeto também contempla noções básicas de responsividade, garantindo uma boa visualização em diferentes tamanhos de tela.

🚀 Tecnologias utilizadas
HTML5
CSS3
🎯 Objetivos do projeto
Praticar a estruturação de páginas com HTML
Aplicar estilização utilizando CSS
Trabalhar com organização de layout (Flexbox)
Implementar conceitos básicos de responsividade
Desenvolver uma interface simples e funcional
📱 Responsividade

O projeto conta com uma media query que adapta o layout para telas menores, priorizando a legibilidade e a experiência do usuário.

💡 Considerações

Por se tratar de um projeto inicial, o foco principal foi o aprendizado e a consolidação dos fundamentos do desenvolvimento front-end. Ainda assim, ele representa um passo importante na evolução das habilidades em criação de interfaces web.
